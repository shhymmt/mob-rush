## Game Features

- **Lane-based movement**: Tap to switch between lanes
- **Auto-shooting**: Units automatically fire at enemies
- **Gates**: Pass through gates to multiply, add, or lose units
- **Boss battles**: Face challenging bosses at the end of each stage

## Controls

- **Mobile**: Tap left/right side of screen to move
- **PC**: Click left/right side of screen to move

## Tech Stack

- Unity 6 LTS
- C#
- WebGL for browser play
